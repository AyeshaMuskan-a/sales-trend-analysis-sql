# Sales Trend Analysis Using SQL

## Overview

This project analyzes sales trends using SQL aggregate functions. The analysis focuses on monthly revenue, order volume, product performance, and overall sales statistics.

This project was completed as part of a Data Analytics Internship Task.

---

## Objective

Analyze sales data to understand:

- Monthly Revenue
- Monthly Order Volume
- Product-wise Revenue
- Product Performance
- Overall Sales Statistics
